# 黎明杀饥模式
## 基础环境说明
本模式为**100档快餐环境**，世界场景、角色属性、各类生物强度统一适配100档数值标准。

## 模组骰子基础设置
1. 模式默认关闭骰子功能，可手动开启
2. 开启骰子模式后，所有韦伯鬼魂将获得**加速、夜视**双重buff增益
3. 骰子使用冷却间隔为100秒
4. 投掷骰子可根据随机点数，召唤三种不同类型的怪物

## 核心玩法总规则
模式分为**人类方**与**鬼方**两大阵营，为对抗玩法。

1. 开局通过roll点划分阵营，在ESC服务器面板使用骰子指令，点数最小的玩家分配为鬼方
2. 单局总人数固定11人，鬼方最多4人，人类玩家初始人数为鬼方三倍
3. 开局阵营战力失衡时，系统会自动提供开局资源补给，管理员配备专属分队辅助插件

### 人类方规则
1. **胜利条件**：全队配合搭配阵容，清空远古区域、击败铁巨人即可获胜（除开军事区）
2. **复活机制**：玩家死亡可复活，但每次复活会永久损失30%最大生命值
3. **专属道具**：可在暗影基座制作彩虹宝石，兑换强心针道具
4. **投降权限**：阵营投降权限仅归属恶魔角色
5. **角色限制**：全队仅可选用一名旺达或恶魔人角色

### 鬼方规则
1. **胜利条件**：通过附身各类生物，击杀全部人类玩家，同时保护铁巨人存活即可获胜
2. **专属机制**：人类方无法复活鬼方玩家

#### 鬼方详细约束与特权规则
1. 阵营固定选韦伯角色，开局需利用蜘蛛自杀完成开局流程
2. 可作祟附身野生生物
3. 禁止摧毁远古科技塔，属于核心禁用操作
4. 生物召唤规则：常规第二天黄昏可输入指令shift+T随机获取生物，前四天无法刷新巨型生物，点数判定规则：点数大于80为巨型生物、点数大于50为精英怪、点数小于50为小型生物
5. 全员自带微光、加速常驻buff
6. 附身生物后，需按F键触发攻击，通过鼠标指向调整攻击目标与方向（可自行深度探索操作细节）
7. 开局七天内，禁止使用恐惧之龙、操控眼球塔守卫
8. 禁止附身单位：龙蝇、梦魇疯猪、远古守护者、阿比盖尔、老麦影人；洞穴场景内额外禁止附身座狼、月亮座狼
9. 特殊召唤机制：达到第六天时，可输入`shift+T`，仅可刷新生成精英生物、巨型生物，shift+Y召唤控制，  
达到第八天时boss血量翻倍，可使用shift+U召唤boss
10. 阵营特权：开局可投票ban掉一名强势角色（例：武神、火女等）
11. 可roll出涟漪单位，但**禁止对其进行附身操作**
12. 场景阶段限制：仅暴动阶段可使用潜伏梦魇，过渡阶段使用会触发游戏bug，严禁操作
13. 时间限制补充：前五天无法获取巨型生物，前七天无法附身眼球塔及眼球塔守卫

（标注粉色内容为暂时弃用规则，暂不生效）

## 战力平衡补充设定
若对局双方战力差距过大，系统将为弱势的人类方补充重生护符或各类基础补给，平衡对局体验。
